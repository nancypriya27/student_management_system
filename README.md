Student Management System
A clean and efficient web application built to manage student records, track information, and streamline administrative workflows. This system features a Python-backed logic layer, a local database structure, and a responsive frontend user interface.

🚀 Features
Student Records Management: Easily add, view, update, and remove student information.

Persistent Storage: Built-in local database handling to ensure data is safely saved across sessions.

Responsive UI: A clean, user-friendly interface designed with HTML, CSS, and interactive JavaScript.

Lightweight Architecture: Quick setup with minimal dependencies.

📁 Project Structure
Plaintext
student_management_system/
├── app.py              # Main application backend logic

└── static/

🛠️ Tech Stack
Backend: Python

Database: SQLite (Local storage)

Frontend: HTML5, CSS3, JavaScript

⚙️ Installation & Setup
Get the project up and running on your local machine by following these steps:

1. Clone the Repository
Bash
git clone https://github.com/nancypriya27/student_management_system.git
cd student_management_system
2. Set Up a Virtual Environment (Optional but Recommended)
Bash
# Windows
python -m venv venv
venv\Scripts\activate

# macOS/Linux
python3 -m venv venv
source venv/bin/activate
3. Run the Application
Bash
python main.py
